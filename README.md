# X-metadata
![X-metadata preview](X-metadata.jpg)

## introduction
X-metadata is a tool for viewing complete metadata from an image or photo

## Instalations
```
$ pkg update -y && pkg upgrade -y
$ pkg install git
$ pkg install python -y
$ pip install colorama
$ git clone https://github.com/Whomrx666/X-metadata.git
$ cd X-metadata
$ python X-metadata.py
```

## Instructions
- **first**: Install tools according to the instructions above
- **second**: Enter the image you want to extract metadata from
- **third**: The tool will start searching and reading the metadata contained in the image
- **last**: And congratulations, the metadata has been found
